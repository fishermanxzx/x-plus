# API

## 属性

| 名称     | 说明               | 类型                                                          | 默认值    |
| -------- | ------------------ | ------------------------------------------------------------- | --------- |
| size     | 尺寸               | enum('default','small','large')                               | 'default' |
| type     | 类型               | enum('default','primary','success','warning','danger','info') | 'default' |
| disabled | 按钮是否为禁用状态 | boolean                                                       | false     |
| plain    | 是否为朴素按钮     | boolean                                                       | false     |
